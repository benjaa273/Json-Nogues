Servicio de Fase Lunar y Coordenadas de Mendoza

Este JSON contiene información astronómica correspondiente a Mendoza, Argentina, obtenida mediante una API pública. Incluye datos de ubicación como nombre del lugar, región, país, latitud, longitud, zona horaria y hora local. También presenta horarios de salida y puesta del Sol y de la Luna. Además, muestra la fase lunar actual en inglés y en español junto con el porcentaje de iluminación visible. Las coordenadas geográficas se organizan en un bloque independiente para facilitar su lectura. Este formato estructurado permite utilizar la información en proyectos educativos, aplicaciones científicas y páginas web relacionadas con astronomía y observación del cielo en tiempo real.
